    Classes

    StationaryModeManager

    Access to current stationary mode state.

    When in stationary mode, no vehicle localization data is available and it is assumed the user is stationary, e.g. when using the system at home. When not in stationary mode, which is usually the default mode, the user is expected to be in a moving vehicle, with localization data being provided by the underlying system.
